Internal feature planning — not public
Shipped
VOD Library ▾
Free/[map]/[agent]

Public browse by map + agent. All VODs from vods table. No account needed.

Pro Match VODs ▾
Free/pro

Pro match VOD library. Best-of-5 support. Managed via public/admin.html.

AI Coach ▾
FreeAIVOD modal

Round-by-round coaching on own VOD notes. Valorant knowledge base injected. 3/day free, 10/day premium. Comms-only detection + canned response.

Coach Chat ▾
FreeAI/coach

Persistent AI chat using general_chat table. Same knowledge base as coach. 5/day free, 20/day premium. Open note always sent as context. Other notes are premium-gated cross-note referencing.

Freeform Notes ▾
Free/coach panel

Collapsible right sidebar on /coach. Autosaves to notebook table. Open note title shown in coach header. Content always sent to AI as context.

VOD Notes ▾
FreeVOD modal

Round-by-round notes per video. Saved to notes table. Shareable via /vod-note/[share_id].

Gameplan + Goals ▾
FreeAI/gameplan

Rank goals with W/L/dodge RR tracking. Projects days-to-goal. AI-generated daily routine. 1 generate/day free, 3/day premium.

Favorites ▾
Free/favorites

Save VODs. Filter by Role/Agent/Map/Player/Sort. Dropdowns built from user's data. Filters sync to URL.

My VODs ▾
Free (10 cap)Unlimited/my-vods

Private VOD library. YouTube/Twitch support. Inline edit (title/map/agent). 10 cap for free, unlimited with Premium.

VOD Playlists ▾
Premium/playlists

Curated ordered VOD study sets. Search library + own VODs. Reorder, play in sequence. Fully premium-gated.

Clip Timestamps ▾
FreeVOD modal

Mark moments in videos while watching. Seeks on click. Stored in vod_marks table. Shows as dots in the progress bar.

Watch Progress ▾
FreeVOD modal

Saves resume position per video in watched table. Progress bar shown on VOD cards.

Lineup Library ▾
Allowlist/util

Top-down map lineups with canvas-drawn agent portrait + ability icon markers. Email allowlist gated. Managed via public/lineup-admin.html.

Premium + Stripe ▾
Billing/premium

Monthly/yearly/lifetime tiers. Stripe Checkout + webhook. PREMIUM_ENABLED master switch. Subscriptions table. Billing portal.

VOD Search ▾
Free/search

Token-based AND match. Order-independent across title/player/map/agent.

Shared Notes ▾
Free/vod-note, /n/[id]

Unlisted link sharing for both VOD notes and notebook entries. No auth required to view. Noindex.

Public VOD Sharing ▾
Free/watch?v=ID

Direct permalink to any VOD. Server-side looks up metadata from the library; falls back to opening by ID. Used as canonical share URL for VOD notes: /watch?v=ID&note=SHARE_ID.

In Progress
Coach page redesign ▾
FreeAI/coach

AI chat is now the primary UI. Notes become a collapsible right panel. Single persistent chat (general_chat). Per-note chat removed. notesLocked bar for premium upsell on cross-note context.

Planned
Pattern Detection ▾
PremiumAI

Cross-note AI analysis to surface recurring mistakes across a user's VOD notes history. Would use the coaching_training_data table or direct note scanning.

Ideas
AI-Curated Playlists ▾
PremiumAI

User describes what they want to study (e.g. "Jett on Bind") and AI builds a custom playlist from the VOD library. Extends the existing playlists feature.

Team Function ▾
Premium

Collaborative teams: shared notes, shared uploaded VODs, members can see/contribute to shared resources. Overlaps with VOD sharing but scoped to a persistent team.

Stats Page ▾
Premium

Agent tier list by ranked winrate per rank, rank distribution, etc. Needs a data source — Riot API or Henrik Dev's unofficial API. Exploratory.

Affiliate Program ▾
Free

Referral links that give users a % of subscription revenue they bring in. Stripe has built-in support for this.

Deadlock Expansion ▾
Free

Expand to cover Deadlock (Valve hero shooter). Likely a separate section similar to the Valorant VOD library. Scope TBD.
